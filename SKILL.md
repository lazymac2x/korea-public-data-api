# Korea Public Data API

**One API for all Korean public data. No keys. No registration. Just call it.**

## What You Get

- **Real-time weather** for 15 Korean cities with 3-day forecasts
- **KRW exchange rates** for 17 currencies, updated every 10 minutes
- **Korean public holidays** for any year (2000-2100)
- **Seoul transit status** — subway lines, bus types, real-time operating info
- **Stock market summary** — KOSPI & KOSDAQ live index data

## Why This Exists

Korean public data is scattered across dozens of government sites, most requiring registration and API keys. This product unifies the most useful data into a single, clean REST API that works out of the box.

## Use Cases

- Build Korean travel apps
- Display KRW exchange rates in fintech products
- Show Korean holidays in calendar apps
- Add Seoul transit info to navigation apps
- Monitor Korean stock market indices
